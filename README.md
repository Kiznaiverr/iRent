# iRent - Aplikasi Sewa iPhone

Aplikasi mobile untuk menyewa iPhone dengan mudah dan praktis.

## Fitur Utama

### Untuk Pengguna
- **Katalog iPhone** - Lihat berbagai model iPhone yang tersedia untuk disewa
- **Pencarian & Filter** - Cari iPhone sesuai kebutuhan Anda
- **Pemesanan Online** - Pesan iPhone langsung dari aplikasi
- **Tracking Order** - Lacak status pesanan Anda secara real-time
- **Manajemen Rental** - Lihat riwayat dan status sewa iPhone Anda
- **Testimoni** - Berikan ulasan dan rating pengalaman Anda
- **Profil Pengguna** - Kelola informasi akun Anda

### Untuk Admin
- **Dashboard** - Monitoring seluruh aktivitas rental
- **Kelola iPhone** - Tambah, edit, dan hapus data iPhone
- **Kelola Pesanan** - Approve dan proses pesanan pelanggan
- **Kelola Rental** - Monitor sewa aktif dan overdue
- **Kelola Pengguna** - Manajemen akun dan verifikasi user

## Cara Menggunakan

### Registrasi & Login
1. Buka aplikasi iRent
2. Klik "Daftar" untuk membuat akun baru
3. Isi data lengkap (Nama, Username, Email, Telepon, NIK)
4. Login dengan username dan password

### Menyewa iPhone
1. Pilih iPhone dari katalog di beranda
2. Lihat detail spesifikasi dan harga sewa
3. Klik "Sewa Sekarang"
4. Pilih tanggal mulai dan akhir sewa
5. Konfirmasi pesanan
6. Tunggu approval dari admin

### Melacak Pesanan
1. Buka menu "Pesanan Saya"
2. Lihat status pesanan (Pending/Disetujui/Selesai/Dibatalkan)
3. Atau gunakan fitur "Lacak Pesanan" dengan kode order

### Mengelola Rental
1. Buka menu "Rental Saya"
2. Lihat daftar iPhone yang sedang Anda sewa
3. Perhatikan tanggal pengembalian untuk menghindari denda

## Ketentuan Penting

- **Verifikasi Akun**: Pastikan NIK dan data Anda valid
- **Denda Keterlambatan**: Pengembalian terlambat akan dikenakan denda
- **Status Akun**: Akun dengan denda belum dibayar akan dinonaktifkan
- **Stok Terbatas**: Periksa ketersediaan stok sebelum memesan
