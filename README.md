## Teknologi

- **Frontend**: Flutter
- **State Management**: Riverpod
- **Networking**: Dio
- **Image Handling**: Cached Network Image, Image Picker
- **Backend**: REST API ([https://backend.nekoyama.my.id](https://backend.nekoyama.my.id))

## Download

Download versi terbaru aplikasi di [GitHub Releases](https://github.com/Kiznaiverr/iRent/releases)

## Struktur Project

```
lib/
├── config/          # Konfigurasi API
├── models/          # Model data
├── providers/       # State management dengan Riverpod
├── screens/         # UI screens
└── services/        # API services
```
